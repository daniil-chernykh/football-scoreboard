# Football World Cup Scoreboard

Простая Java-библиотека для отображения и управления результатами матчей Чемпионата мира по футболу.

## Требования

- Java 8 или выше
- Maven (для сборки)

## Установка

1. Клонируйте репозиторий:
   ```bash
   
   cd football-scoreboard
   ```
2. Сборка проекта:
    ```bash
   mvn clean install
   ```# football-scoreboard
